How It Works

From camera to arrow in under a second

Four automatic steps happen every time you lift your phone — no interaction needed.

STEP 01

Capture

Camera snaps a JPEG frame at quality 0.7, max 1280×720, under 2 MB. Sent to the AI server over HTTPS.

→
STEP 02

Identify

DINOv2 ViT-B/14 generates a 768-d embedding. FAISS retrieves top-5 matches. Majority vote → location + confidence.

→
STEP 03

Route

A* algorithm computes the shortest path from your node to the destination, returning ordered waypoints with coordinates.

→
STEP 04

Guide

AR arrow on camera feed points toward the next waypoint. Mini-map updates. You walk. Loop repeats every 3.5 seconds.

Facebook AI Similarity Search. Exact cosine similarity. Sub-millisecond retrieval on normalised embedding vectors.

PyTorch Image Models library. ViT-B/14 pretrained on LVD-142M (142M curated images). Transfer learning backbone.

ML Framework

Python Flask

REST API with 7 endpoints. HTTPS with SSL. Threaded gunicorn workers. In-memory image handling — no disk writes.

Backend

A* Pathfinding

Weighted graph, Euclidean + floor-penalty heuristic. Computes optimal routes across all 13 campus nodes.

Algorithm

Canvas 2D API

Native browser rendering. AR arrow overlay at 30 Hz. Mini-map drawn from node coordinate graph — zero dependencies.

Benefits

Why Vision Navigator over everything else

Traditional solutions require beacons, apps, or infrastructure. We need none of it.

No Hardware Cost

Bluetooth beacons cost thousands per building. Vision Navigator uses the camera already in every student's pocket.

Deploy in 48 Hours

No physical installation. Collect photos, retrain the model, update the graph. A new building goes live in under two days.

Any Smartphone, Any Browser

Chrome on Android. Safari on iPhone. Share a link — navigation starts in seconds without any app store visit.

Privacy by Design

Frames processed in-memory. No images stored, no user tracking, no movement database. GDPR-friendly by default.

No GPS Dependency

GPS fails inside buildings. Visual recognition and gyroscope work on every floor, in every corridor, regardless of signal.

Add more training images to the FAISS index and accuracy increases — no full retraining required. The system learns your space.
